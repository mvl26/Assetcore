# QMS Mapper — Examples

## Example 1: IMM-05 Asset Document Repository

**Input:** `IMM-05`

**QMS Document Mapping:**

| Level | Document | Notes |
| --- | --- | --- |
| QC | QC-05: Document Control Policy | Visibility rules, retention |
| PR | PR-05-01: Asset Document Upload Procedure | Steps for upload → review → approve |
| WI | WI-05-01: How to classify doc_category | Technical/Legal/Certification guidance |
| BM | BM-05-01: Asset Document Upload Form | Required fields checklist |
| HS | HS-05-01: Asset Document Record | Auto-generated by system |
| KPI | KPI-05-01: Document Completeness % | Per asset, per facility |

**CAPA Triggers:**

```text
TRIGGER: Document expires without renewal
CAPA Type: Corrective
Severity: Major
Owner: Tổ HC-QLCL
Response SLA: 5 business days
```

**ISO 13485 Clauses:** 4.2.4 (control of records), 7.5.1 (service control)

## Example 2: IMM-09 Repair (Corrective Maintenance)

**CAPA Triggers:**

```text
TRIGGER: Same asset fails 3× in 90 days
CAPA Type: Preventive
Severity: Critical
Owner: Biomed Engineer + Workshop Head

TRIGGER: Repair exceeds SLA by 24h
CAPA Type: Corrective
Severity: Major
Owner: Workshop Head
```

**KPIs:**

| KPI | Formula | Target |
| --- | --- | --- |
| MTTR | Avg(repair_close - repair_open) | ≤ 48h |
| Repeat Failure Rate | repairs with same root cause / total | < 10% |
| First-Time Fix Rate | resolved without re-open / total | > 85% |

## Example 3: IMM-11 Calibration

**Regulatory mapping:**

- TT46/2017 Điều 5: Kiểm định bắt buộc hàng năm
- ISO 13485:2016 §7.6: Calibration records 5-year retention
- NĐ98/2021 Điều 12: Hết hạn kiểm định = ngừng sử dụng

**Change Control:** Required before changing calibration frequency or tolerance
